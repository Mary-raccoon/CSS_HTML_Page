1) Create a new repository on GITHUB and call it GitHubRepoAssignment 
2) Open your terminal/command prompt, and create a GitHubRepoAssignment folder - mkdir GitHubRepoAssignment                                                     
3) Initialize a git repository in your GitHubRepoAssignment directory - git init
4) Make a readme.md in your GitHubRepoAssignment folder explaining its contents. - touch readme.md
5) Do a status check to see the changes made, then add, commit and push your changes - 
git status
open readme.md
git add readme.md
git commit -m "my new changes"
